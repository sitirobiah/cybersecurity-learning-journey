# Kali Linux Setup on Virtual Machine

## 🎯 Objective
Install and run Kali Linux on a virtual machine for cybersecurity learning.

## 🛠️ Tools
- [VirtualBox](https://www.virtualbox.org/) or [VMware Workstation Player](https://www.vmware.com/products/workstation-player.html)  
- [Kali Linux ISO](https://www.kali.org/get-kali/)

## 📌 Steps
1. Download Kali Linux ISO from the official website  
2. Install VirtualBox or VMware Workstation Player  
3. Create a new virtual machine  
4. Allocate resources:  
   - RAM: at least 4GB  
   - Storage: at least 40GB  
5. Install Kali Linux following the on-screen instructions  
6. Start the virtual machine  

## 💡 Insights
- Following official documentation ensures installation success and avoids errors
-  Using prebuilt VM images is faster and reduces setup errors
